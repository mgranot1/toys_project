Toys Project

User routes:

METHOD URL BODY DESCRIPTION
POST (register) /users 
{
    "fullName": { "firstName": "Eva", "lastName": "Williams" },
    "email": "eva.williams@example.com",
    "password": "evapassword",
    "role": "admin"
}
Add a new user to the data base ,if you want to see the template of valid user.
POST (login) /users/login 

                        
{
    "email": "eva.williams@example.com",
    "password": "evapassword"
}
User login, if the email and password are correct the server will respond with a token, if you want to see the template of valid user.
DELETE /users/delete/:idDel none Delete a user from the data base.
PUT /users/:idEdit 
{
    "fullName": { "firstName": "Eva", "lastName": "Williams" },
    "email": "eva.williams@example.com",
    "password": "evapassword",
    "role": "admin"
}
Update user details and save to databace.

Toys routes:

METHOD URL BODY DESCRIPTION
GET /toys none Get a list of all the toys, will present 10 results per page.
GET /toys/search/s=Toy1 none search for a toy by searching follwing name or info after the "s=", it will present 10 results per page.
GET /toys/category/:catName none Get a list of all the toys in toys category, it will present 10 results per page.
POST /toys 
{
    "name": "Teddy Bear",
    "info": "A cuddly companion for kids of all ages.",
    "category": "Plush",
    "img_url": "https://example.com/teddy.jpg",
    "price": 14.99
}
Add a new toy to the data base, You need a token for this query , if you want to see the template of valid toy
PUT /toys/ID 
{
    "name": "Teddy Bear",
    "info": "A cuddly companion for kids of all ages.",
    "category": "Plush",
    "img_url": "https://example.com/teddy.jpg",
    "price": 14.99
}
Update a toy in the data base,You need a token for this query.
DELETE /toys/ID none Delete a toy from the data base, You need a token for this query.
